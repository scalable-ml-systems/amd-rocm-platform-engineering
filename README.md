# AMD Instinct Production AI Platform Engineering Lab

Production qualification and architecture-aware inference engineering for AMD Instinct GPUs.

## Platform Scope

- ROCm and HIP
- MI300X partitioning and topology
- xGMI and RCCL
- AITER and ATOM
- vLLM and SGLang
- DeepSeek MLA and MoE
- FP8 inference
- Kubernetes GPU Operator and DRA
- Kueue admission
- AMD-native observability and remediation
- Two-node RDMA distributed serving
